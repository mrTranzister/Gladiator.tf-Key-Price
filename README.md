# Gladiator.tf-Key-Price
Displays the global key price at which Gladiator.tf bots trade on the Backpack.tf.

# Installation
1. Install <a href="https://www.tampermonkey.net/" target="_blank" rel="noopener noreferrer">Tampermonkey</a> browser extension
2. Click <a href="https://github.com/mrTranzister/Gladiator.tf-Key-Price/raw/refs/heads/main/Gladiator.tf%20Key%20Price-1.8.user.js" target="_blank" rel="noopener noreferrer"> here to install the script</a> or copy the code manually
3. To get the key price, you need to click the refresh icon. The key price updates automatically every 15 minutes. The timer shows how long it’s been since the last key price update.
